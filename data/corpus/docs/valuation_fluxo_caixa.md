# Fluxo de caixa descontado sintetico

Este texto sintetico apresenta um fluxo de caixa descontado de forma conceitual.
As premissas sao ficticias e servem apenas para validar o pipeline local de
ingestao do Agent-0.

Um modelo robusto separa crescimento, margem, reinvestimento e custo de capital.
O resultado deve ser interpretado como intervalo, nao como numero absoluto ou
recomendacao operacional.
