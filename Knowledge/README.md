# 📚 Knowledge

## Structure

```
Knowledge/
├── Radiology/        # Ultrasound, Doppler, MSK, CT, MRI concepts
├── AI-ML/            # Machine learning, deep learning, LLM theory
├── Clinical/         # Clinical protocols, guidelines, evidence
└── Mental-Models/    # Cross-domain thinking frameworks
```

## Conventions

- One file per concept (atomic notes)
- Frontmatter with `tags`, `date`, `source`, `domain`
- Link to related notes and projects with relative paths
- Evidence quality rated where applicable: `[RCT]`, `[Meta]`, `[Expert]`, `[Guideline]`
