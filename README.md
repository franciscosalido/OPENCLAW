## What is OPENCLAW?

OPENCLAW is the personal knowledge operating system of [Francisco Salido](https://github.com/franciscosalido), a radiologist and AI specialist (MBA in AI). It serves as a living repository connecting:

- **Radiology** — Ultrasound, Doppler, MSK, and AI-augmented imaging
- **AI/ML** — Applied machine learning, LLM workflows, prompt engineering
- **Claude Ecosystem** — Claude Desktop, Cowork, Claude Code integrations
- **Research** — Curated literature, notes, and clinical evidence synthesis

This repo is built to be queried, extended, and automated — not just read.

## Philosophy

> *"Don't just collect information — build a system that thinks with you."*

OPENCLAW is designed around three principles:
1. **Capture** — Every insight, paper, or workflow gets a place
2. **Connect** — Notes link to projects; research feeds workflows
3. **Execute** — Everything is actionable, not just archival
