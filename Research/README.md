# 🔬 Research

Literature reviews, clinical evidence synthesis, and annotated papers.

## Structure

```
Research/
├── Radiology/        # Imaging AI, ultrasound, MSK papers
├── AI-ML/            # ML/DL methods, LLM papers
├── Clinical/         # Clinical guidelines, RCTs, meta-analyses
└── Reviews/          # Systematic reviews and evidence synthesis authored here
```

## Evidence Tagging System

| Tag | Meaning |
|-----|---------|
| `[RCT]` | Randomized controlled trial |
| `[Meta]` | Meta-analysis or systematic review |
| `[Guideline]` | Official society guideline |
| `[Expert]` | Expert opinion / editorial |
| `[Preprint]` | Not yet peer-reviewed |
| `[Bias: X]` | Known bias type flagged |

## Conventions

- File format: `YYYY-AuthorName-ShortTitle.md`
- Include: DOI, abstract, key findings, limitations, relevance to OPENCLAW projects
- Flag conflicts of interest and funding sources where known
