# ⚙️ Workflows

AI-assisted automation, prompt templates, Claude integrations, and reusable pipelines.

## Structure

```
Workflows/
├── Prompts/          # Reusable prompt templates (Claude, GPT, etc.)
├── Claude/           # Claude Code, Cowork, Desktop integrations
├── Scripts/          # Python/Node automation scripts
└── Pipelines/        # End-to-end AI workflows (RAG, agents, etc.)
```

## Claude Ecosystem Integration

OPENCLAW is designed to work natively with:
- **Claude Desktop** — conversational interface for knowledge retrieval
- **Claude Cowork** — document and task automation
- **Claude Code** — codebase-level AI assistance
- **MCP Servers** — GitHub, Calendar, PubMed integrations

## Prompt Conventions

- One file per prompt template
- Include: purpose, input variables, expected output, example
- Version with date prefix: `v1-YYYY-MM-DD`
- Tag with domain: `[Radiology]`, `[Research]`, `[Coding]`, `[Analysis]`
